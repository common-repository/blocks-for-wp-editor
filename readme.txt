== Description ==
**Blocks for WP Editor** allows you to add blocks in wordpress gutenberg editor.

Check some of its features:

1. Posts block by type - using this block we can show posts by type, author, categories etc.
[youtube https://youtu.be/WrTAMJu89ns]

== Installation ==
1. Unzip the downloaded zip file.
2. Upload the plugin folder into the `wp-content/plugins/` directory of your WordPress site.
3. Activate "Blocks for WP Editor" from Plugins page.
